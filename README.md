# retale-portfolio


## Présentation du projet

Ce projet est un portfolio professionnel conçu pour présenter mon profil, mes compétences et mes projets en développement web.

L’objectif est de créer un site moderne, responsive et accessible, servant de vitrine professionnelle.

## Fonctionnalités

### Hero Section
- Présentation claire
- Pitch personnel
- Bouton d’appel à l’action

### À propos
- Présentation de mon parcours
- Évolution vers le développement web
- Mise en avant de mes compétences

### Projets
- Présentation de projets réalisés
- Aperçu des projets


## Stack utilisée

- HTML5
- Tailwind CSS
